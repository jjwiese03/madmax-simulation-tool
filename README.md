## Project description 

This simulation tool was developed as part of a bachelor’s thesis at RWTH Aachen University ([link](https://www.institut3a.physik.rwth-aachen.de/global/show_document.asp?id=aaaaaaaaddzwdlm)). The purpose of the tool is to visualize key properties of the MADMAX experiment as a function of the experimental settings.

## Status

The tool is fully functional and currently supports simulations of both the **boost factor** and the **reflectivity**.
