▸ draw(axis, discs)

Draws the plot onto the canvases.

When axis is true, the axis line, ticks, tick labels and axis label are (re-)drawn onto the axis canvas.

When discs is true, all discs, the gap arrows and labels between them, the marquee selection rectangle (if active) and the mirror indicator (if enabled) are (re-)drawn onto the disc canvas.

Disc

Represents a single disc within a DiscCollection.

A disc has a position, a width, a dielectric constant (epsilon) and can be selected. Discs are kept in a sorted array inside their parent DiscCollection and know their own index within that array.

new Disc(collection: DiscCollection, position: number, width: number, epsilon: number, selected: boolean, index: number)
Parameters
collection (DiscCollection) The collection to which the disc belongs.
position (number) The x-coordinate of the disc.
width (number) The width of the disc.
epsilon (number) The dielectrical constant of the disc material.
selected (boolean) Whether the disc is selected.
index (number = null) The index of the disc within its collection.

▸ binSearch(position)

Implements binary search to find the index of the disc that is on the right side of the given position. If position is greater than the position of the last disc, it returns the last disc's index. Used in DiscCollection#indexOf and DiscCollection#positionNeighbour.

Time complexity: O(log(n)) (n = number of discs in DiscCollection).

IMPORTANT: binSearch(x) for any x greater than lastDisc.position returns lastDisc.index.

binSearch(position: number): number
Parameters
position (number) The position to search for.
Returns
number: The index of the disc to the right of position .
Example
// DiscCollection consists of Discs with positions: 2, 3, 4, 5, 6
binSearch(4.5); // => 3
binSearch(4);   // => 2
binSearch(2.5); // => 1
binSearch(100); // => 4
